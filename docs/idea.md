# 想法与需求（整理自 `docs/idea.txt`）

## 需求

- 消毒
- 便携性
- 收纳体积
- 可调定时消毒
- 人体靠近保护
- 远程遥控
- 远程数据传输

## 产品设计要点

- 外壳材料：PP 塑料（耐高温、适用于食品容器）

## 设计想法（Idea）

1. 消毒灯与电源做成可拆模块，便于清洗，拆掉后可作为普通收纳盒使用
2. 智能家居接入（例如米家 / Home Assistant）
3. 磁吸结构
4. 紫外线前端（消毒灯）
5. 臭氧发生器
6. 内部反射、外部屏蔽

## 硬件清单（草稿）

- 主控：Pi Zero W / ESP-WROOM-32（ESP-32S）
- 人体红外感应：RD-624
- 显示：0.91 寸 I2C OLED（白色）
- 驱动：三极管 S8050（NPN）/ MOS 管 5N10 / AO3442 等
- LED 灯珠：6 颗
- 电源：聚合物锂电池 + 充放电管理模块 + 总电源开关
- 交互：确认/上/下 按键

## 软件

- Blinker
- Home Assistant
